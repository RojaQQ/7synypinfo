Мәтіндік файлдар

Алдыңғы сабақтарда біз мәтіндік файлдармен жұмыс істеуді үйрендік (ашу, оқу, жазу).

Бинарлық файлдар

Python-да бинарлық файлдармен жұмыс істеу үшін open() функциясына 'b' режимін қосамыз (мысалы, 'rb' – бинарлық оқу).

Ескерту:

7-сыныпта біз негізінен мәтіндік файлдардағы арнайы функцияларды қарастырамыз.

1. readline() функциясы

Анықтамасы: Файлдағы жолдарды жеке-жеке оқу үшін қолданылады.

  • readline() (жақша іші бос)

    Бір жолды толығымен оқиды (немесе жолдың қалғанын).

  • readline(n) (жақшада сан тұрса)

    Жолдың алғашқы n символын ғана оқиды.

Маңызды: Файлдан оқыған деректерді жаңа жолға жазғанда '\n' (жаңа жол) символы ескерілуі керек.

2. seek() функциясы

Қызметі

Файлдағы меңзерді (курсорды) қажетті орынға ауыстыру.

Синтаксис

file.seek(n)

мұндағы n – меңзер тұратын орын нөмірі (байтпен).

Айырмашылығы

read() → Файлды ретімен, тұрған жерінен оқиды.

seek(n) → Файлдың басынан санағанда n-ші орынға секіреді.

1-Мысал (readline қолдану)

Тапсырма:

Input.txt файлының 1-жолында әріптер, 2-жолында сандар берілген. 1-жолдың алғашқы 10 әрпін және 2-жолдың алғашқы 5 цифрын оқу керек.

# Input.txt файлы алдын-ала құрылған деп есептеледі f = open("Input.txt", "r") p = f.readline(10) # 1-жолдан 10 символ оқу f.readline() # 1-жолдың қалғанын оқып, келесі жолға өту h = f.readline(5) # 2-жолдан 5 символ оқу print(p) print(h) f.close()

2-Мысал (seek және нәтиже)

Тапсырма: Input.txt файлындағы "abcdef..." жолының 2-орнынан бастап 15 әріпті '*' символымен ауыстыр.

f = open("Input.txt", "w+") # w+ режимі f.write("abcdefghijklmnopqrstuvwxyz") f.seek(2) # Меңзерді 2-ші орынға қою f.write("*" * 15) # 15 жұлдызша жазу f.close()

Нәтижесі (Output):

Бастапқы:
abcdefghijklmnopqrstuvwxyz
Өзгерген соң:
ab***************rstuvwxyz

(Алғашқы 'ab' [0 және 1 индекстер] қалады)

1-Практикалық тапсырма

Тапсырма (Оқулық, 113-бет): Үш, төрт және бес таңбалы сандар тізбегінен қосындыны табу. Сандар арасы бос орынмен бөлінген деп есептейміз.

Мысалы: 100 2000 30000 (Аралықтарында 1 бос орын бар)

1. read(3): 1-ші санды (3 таңбалы) оқимыз.
2. seek(4): Бос орынды тастап, 4-ші орынға секіреміз.
3. seek(9): Келесі бос орынды тастап, 3-ші санға секіреміз.
fin = open("Input.txt", "r") a = int(fin.read(3)) # "100" оқылады fin.seek(4) # " " (бос орын) аттап өтіледі b = int(fin.read(4)) # "2000" оқылады fin.seek(9) # Келесі бос орын аттап өтіледі c = int(fin.read(5)) # "30000" оқылады print(a + b + c) fin.close()

2-Практикалық тапсырма

Тапсырма (Оқулық, 114-бет, 3-тапсырма):

Input.txt файлында a, b, c, d және p сандары бір жолда, бос орын арқылы берілген.

Есептеу: ((a + b) % p) * ((c - d) % p)

f = open("Input.txt", "r") line = f.readline() # Жолды тұтас оқу: "25 18 45 12 9" f.close() s = line.split() # Бос орын бойынша бөлу: ['25', '18', '45', '12', '9'] a = int(s[0]) b = int(s[1]) c = int(s[2]) d = int(s[3]) p = int(s[4]) res = ((a + b) % p) * ((c - d) % p) f_out = open("Output.txt", "w") f_out.write(str(res)) f_out.close()

Сабақ қорытындысы

readline(n)

Файлдан белгілі бір мөлшердегі символдарды оқуға ыңғайлы құрал.

seek(n)

Файл ішінде "еркін қозғалуға", яғни қалаған орыннан оқуға немесе жазуға мүмкіндік береді.
